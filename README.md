# 暮雪の旅

我们未曾挣脱地心引力，也无法让逝去的时间倒流。

但在这个故事里，有一位魔法师，她的足迹遍布过去，现在，未来，以及每一个时间线的岔路。

>_派一艘飞船，去往星辰_
>_它的归途是深空_
>
>_遣一位骑士，奔赴战场_
>_他的传说在歌谣_
>
>_念一段咒文，祈万千神明_
>_祂的终点是遗忘_
>
>_翻一页时间，窥万千路线_
>_她的故事尚未终结_

![8df44ffcc2fa9993f6d33800944355b5](https://github.com/user-attachments/assets/1a3048d8-a5d4-447c-b417-7fb85231eb7e)


这是一个具有实验性质的小说系列，借助“可跨越时间线”的设定，涉猎不同大量世界观，题材和文风的故事，每个故事都相对独立，长度从短篇到长篇小说不等。
